# urban-mobility (Lagos Traffic Story)

## Motivation

Every city has its own transportation identity.

Most computer vision datasets focus on vehicles commonly found in Europe, North America, or East Asia. These datasets often overlook transportation systems unique to cities like Lagos, Nigeria.

Lagos presents a distinctive traffic environment, including Danfo buses, Korope minibuses, Keke Napep tricycles, Okada motorcycles, BRT buses, Molue buses, delivery vehicles, heavy trucks, and emergency services.

This project aims to develop an AI-powered urban mobility pipeline tailored to Lagos by combining:

- Computer Vision
- Machine Learning
- Road Network Analysis
- SUMO Traffic Simulation
- Supply Chain Analytics

The objective is to understand how traffic patterns influence congestion, urban logistics, delivery efficiency, and sustainable supply chain management.

Ultimately, this research seeks to contribute datasets, models, and tools that better represent transportation systems in emerging cities.
